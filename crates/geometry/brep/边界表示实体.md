# 边界表示实体 (Brep)

## 1. Brep (Boundary Representation)
完整的边界表示拓扑结构，用于构建复杂实体。

### 1.1 拓扑元素
*   **BrepFace (面)**:
    *   关联几何: `Surface`。
    *   关联边界: `BrepLoop`。
*   **BrepLoop (环)**:
    *   区分外环 (Outer) 与内孔 (Inner)。
    *   由有序的 `BrepTrim` 组成。
*   **BrepTrim (剪切边)**:
    *   定义在 2D UV 参数空间。
    *   关联几何: `Curve2D`。
*   **BrepEdge (物理边)**:
    *   定义在 3D 空间。
    *   关联几何: `Curve3D`。
*   **Vertices (顶点)**: 拓扑顶点。

### 1.2 几何关联
Brep 同时维护两套几何数据以支持精确计算与显示：
*   **空间几何**: `Curve3D`, `Surface`。
*   **参数几何**: `Curve2D` (用于裁剪曲面)。

### 1.3 显示
*   **DisplayValue**: 包含 `MeshXYZ` 列表，用于渲染管线显示。
