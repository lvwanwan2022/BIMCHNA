# 核心算法 (Core Algorithms)

## 1. 数学引擎 (EQN)
提供了一套代数方程求解器，用于几何求交和数值计算。

### 1.1 EQN1d
一元多项式方程类。
*   **功能**: 构造线性方程、高次方程。
*   **运算**: 求值、求导、求根（包括二次方程解析解）。

### 1.2 EQN2d
二元方程组类。
*   **功能**: 联立两个方程进行求解。
*   **应用**: 处理直线、圆等几何方程的解析表达，支持牛顿迭代法求交点。

## 2. 几何运算

### 2.1 变换 (Transformation)
*   **接口**: `ITransformable`。
*   **操作**:
    *   平移 (Translation)。
    *   旋转 (Rotation): 基于罗德里格公式 (Rodrigues’ Rotation Formula)。
    *   缩放 (Scale)。

### 2.2 求交 (Intersection)
*   线线求交。
*   线面求交。
*   面面求交。
*   **输出**: `IntersectPoint` (交点), `IntersectLine` (交线)。

### 2.3 投影与距离
*   点到平面的投影与距离。
*   点到直线的投影与距离。
*   向量的正交分解与垂直判断。
